# UnB FCTE - Software Product Engineering (FGA0316) 2026.1
## High-Performance Software Factory | AI-First Methodology

Welcome to the official GitHub Organization for the **Software Product Engineering** course at the **University of Brasília (UnB)**, Faculty of Gama (FGA). 

This semester, we are operating as a high-performance software factory, developing critical applications for the **Civil Police of the Federal District (PCDF)**. Our focus is on strategic product lifecycle management, rigorous security (DevSecOps), and AI-assisted development.

---

### 🚀 The "AI-First" Mission
We utilize **GitHub Copilot** as a primary coding accelerator. Students act as **Solution Architects** and **Human-in-the-loop Auditors**, ensuring that AI-generated code meets the high security and legal standards required by law enforcement systems.

### 🛡️ DevSecOps Ecosystem
Every repository in this organization is integrated into a robust CI/CD pipeline via **GitHub Actions**, including:
* **Static Analysis (SAST):** Bandit (Python security) & SonarQube (Quality Gate).
* **Supply Chain Security (SCA):** Safety (Dependency scanning).
* **Testing:** Cypress (E2E) and Pytest (Unit/Integration).
* **Observability:** Sentry & Grafana.

### 📁 Projects for PCDF (2026.1)
1. **Chain of Custody:** Digital evidence integrity (SHA-256/WORM).
2. **Seizure Assistant:** Mobile OCR for instant IMEI/Serial extraction.
3. **OSINT Searcher:** Investigative web scraping & dossiers.
4. **Semantic Analyzer:** NLP/RAG for confidential testimony processing.

---

### 🏛️ Governance & Frameworks
We bridge the gap between agility and formal governance:
* **PRINCE2:** Stage boundary management and End Stage Reports.
* **PMBOK 6/7:** Scope, Risk, and Earned Value Management (EVM).
* **Agile:** Sprints, Backlog management (ZenHub), and Lean Inception.
